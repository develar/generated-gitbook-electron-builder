CommonLinuxOptions

Kind: interface of electron-builder
Properties

  • synopsis String - The short description.
  • description String - As description from application package.json, but allows you to specify different for Linux.
  • category String - The application category.
  • desktop any - The Desktop file entries (name to value).

CommonNsisOptions

Kind: interface of electron-builder
Properties

  • unicode = true Boolean - Whether to create Unicode installer.
  • guid String - See GUID vs Application Name.
  • warningsAsErrors = true Boolean - If warningsAsErrors is true (default): NSIS will treat warnings as errors. If warningsAsErrors is false: NSIS will allow warnings.
