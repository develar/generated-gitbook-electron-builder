AppImageOptions

The top-level appImage key contains set of options instructing electron-builder on how it should build AppImage.

  • systemIntegration = ask "ask" | "doNotAsk" - The system integration installation.

Inherited from CommonLinuxOptions:

  • synopsis String - The short description.
  • description String - As description from application package.json, but allows you to specify different for Linux.
  • category String - The application category.
  • desktop any - The Desktop file entries (name to value).

Inherited from TargetSpecificOptions:

  • artifactName String - The artifact file name template.
  • publish The publish options.

LinuxTargetSpecificOptions

The top-level apk, freebsd, pacman, p5p,rpm keys contains set of options instructing electron-builder on how it should build corresponding Linux target.

  • depends Array<String> - Package dependencies.
  • icon String
  • packageCategory String - The package category.
  • vendor String
  • maintainer String
  • afterInstall String
  • afterRemove String

Inherited from CommonLinuxOptions:

  • synopsis String - The short description.
  • description String - As description from application package.json, but allows you to specify different for Linux.
  • category String - The application category.
  • desktop any - The Desktop file entries (name to value).

Inherited from TargetSpecificOptions:

  • artifactName String - The artifact file name template.
  • publish The publish options.
