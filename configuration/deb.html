Debian Package

The top-level deb key contains set of options instructing electron-builder on how it should build Debian package.

  • compression = xz "gz" | "bzip2" | "xz" - The compression type.
  • packageCategory String - The package category.
  • priority String - The Priority attribute.
  • depends Array<String> - Package dependencies. Defaults to ["gconf2", "gconf-service", "libnotify4", "libappindicator1", "libxtst6", "libnss3"].

Inherited from LinuxTargetSpecificOptions:

  • icon String
  • vendor String
  • maintainer String
  • afterInstall String
  • afterRemove String
