MAS

The top-level mas key contains set of options instructing electron-builder on how it should build MAS (Mac Application Store) target. Inherits macOS options.

  • entitlements String - The path to entitlements file for signing the app. build/entitlements.mas.plist will be used if exists (it is a recommended way to set). Otherwise default.
  • entitlementsInherit String - The path to child entitlements which inherit the security settings for signing frameworks and bundles of a distribution. build/entitlements.mas.inherit.plist will be used if exists (it is a recommended way to set). Otherwise default.
  • binaries Array<String> - Paths of any extra binaries that need to be signed.
